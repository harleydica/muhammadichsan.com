Barcode dan QR Code sepasang adik kakak yang kembar namun tidak identik, kita sering menemukanya di banyak tempat, biasanya barcode digunakan untuk produk jualan dan qr code untuk pembayaran digital seperti qris contohnya.

Banyak orang yang salah kaprah dengan dua mahluk ini, banyak yang mengira bahwa code tersebut merupakan gambar karena kita diwajibkan untuk melakukan scan untuk menggunakanya. Padahal barcode dan qr code itu hanyalah representasi dari suatu kumpulan binary saja, ya seperti hexadecimal maupun decimal namun bedanya disini kita menggunakan **visualisasi**.

Oke langsung saja kita ke pembahasanya.
